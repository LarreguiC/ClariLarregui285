What Constitutes a creative web project today?

A creative web project can be about any topic. It can tell a story, an idea or it can bring on a discussion. These discussions can vary from the arts, a performance, political, social issues and even more. With every event that occurs in our world a web project can bring on an impact in order to create positive changes. Every web project always has a goal to be achieved, and most web projects do rely on on viewers who are persuading to take a small part of that project in order to make it happen. A successful a creative web project can happen if a creative team is able to capture the eye of the viewer and is able to appeal them in a unique way. This can happen if the web page design is appealing, if the goal of the project is for the greater good and so much more.
